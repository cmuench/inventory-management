System Architecture

Browser — Vue 3 SPA (port 3000)
  • FilterBar.vue — global filter controls (Period · Location · Category · Status)
  • 6 route views: Dashboard · Inventory · Orders · Demand · Spending · Reports
  • Shared state via useFilters() composable — single source of truth for filters
  • useAuth() — mock user session · useI18n() — en/ja locale + USD/JPY currency
  • api.js — Axios client; builds URLSearchParams from active filter state
HTTP GET · query params · CORS *
FastAPI Server — Python (port 8001)
  • Route handlers in main.py — 15+ endpoints across inventory, orders, demand, spending, reports
  • apply_filters() — warehouse, category, status (case-insensitive string matching)
  • filter_by_month() — individual month (2025-01) or quarter (Q1-2025) expansion
  • Pydantic v2 models validate all responses before serialisation
  • POST /api/purchase-orders — only mutating endpoint; appends to in-memory list
loaded once at startup
mock_data.py
  • Reads all JSON files at import time
  • Exposes module-level lists (orders, inventory, …)
  • purchase_orders list is mutable (POST appends)
server/data/ — JSON files
  • inventory.json · orders.json
  • demand_forecasts.json · backlog_items.json
  • spending.json · transactions.json
  • purchase_orders.json (starts empty)

Filter System

warehouse
all · San Francisco · London · Tokyo
category
all · Circuit Boards · Sensors · Actuators · Controllers · Power Supplies
status
all · delivered · shipped · processing · backordered
month
all · 2025-01 … 2025-12 · Q1-2025 … Q4-2025
Filter propagation path
1
FilterBar.vue — user interaction
4 <select> dropdowns bound to useFilters() refs via v-model
2
useFilters.js — singleton composable
selectedPeriod · selectedLocation · selectedCategory · selectedStatus (ref)
3
View components — reactive watch
watch([...filterRefs], () => loadData()) — any change triggers reload
4
api.js — getCurrentFilters()
Omits 'all' values; builds ?warehouse=London&status=shipped query string
5
FastAPI — apply_filters() + filter_by_month()
Case-insensitive matching on in-memory lists; quarter expanded to 3 months
6
Note: Inventory endpoint ignores month
Inventory has no time dimension — month param silently dropped
